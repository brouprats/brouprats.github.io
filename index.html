Univariate LSTM Time Series Python

Predicció Univariada de Preus Univariate Stock Price Prediction

Model LSTM que utilitza exclusivament el preu de tancament com a variable d'entrada. Captura la dinàmica temporal intrínseca de la sèrie sense informació addicional, servint com a línia base per comparar amb el model multivariant. LSTM model using exclusively the closing price as input variable. Captures the intrinsic temporal dynamics of the series without additional information, serving as a baseline to compare with the multivariate model.

ENFOCAMENT APPROACH

Una única sèrie temporal com a input: el preu de tancament ajustat. El model aprèn patrons de memòria a llarg termini sense features externs. A single time series as input: the adjusted closing price. The model learns long-term memory patterns without external features.

Multivariate LSTM Feature Eng. Python

Predicció Multivariada de Preus Multivariate Stock Price Prediction

Extensió del model univariant que incorpora múltiples variables d'entrada: indicadors tècnics (RSI, MACD, Bollinger Bands), volum i dades de mercat. Permet al model contextualitzar el preu dins un entorn financer més ric. Extension of the univariate model incorporating multiple input variables: technical indicators (RSI, MACD, Bollinger Bands), volume and market data. Allows the model to contextualize the price within a richer financial environment.

HIPÒTESI HYPOTHESIS

Afegir indicadors tècnics com a features millora la capacitat predictiva respecte al model univariant, especialment en períodes d'alta volatilitat. Adding technical indicators as features improves predictive capacity over the univariate model, especially during high-volatility periods.
