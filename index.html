Predicció d'accions amb LSTM

Deep Learning Time Series Python Quant Finance TFG

Xarxes Long Short-Term Memory per a la predicció de preus borsaris

Disseny i implementació d'un model de deep learning per predir el preu de tancament d'accions del mercat americà. El model captura dependències temporals a llarg termini que models clàssics com ARIMA no poden representar adequadament.

HIPÒTESI DE RECERCA

Les sèries temporals financeres contenen patrons de memòria a llarg termini que les LSTM poden aprendre sense feature engineering manual exhaustiu.

Les dades d'entrada inclouen preu de tancament, volum i indicadors tècnics (RSI, MACD, Bollinger Bands) normalitzats amb MinMaxScaler. L'entrenament utilitza seqüències de 60 dies per predir t+1.
